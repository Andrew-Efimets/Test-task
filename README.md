Защита данных: Шифрование персональных данных и сообщений (AES-256) на уровне Eloquent-кастов, а для мгновенного поиска по зашифрованной базе данных настроены хэш-индексы (для почты email_blind, чтобы идентифицировать email пользователя).

Функционал чатов: Приватные диалоги (с автопоиском дубликатов) и групповые чаты с управлением участниками и полноценным real-time циклом сообщений (отправка, редактирование и мягкое удаление через Laravel Reverb).

Структура: Код разнесен по дополнительным папкам: вся бизнес-логика и транзакции в сервисах (app/Services/Chats), валидация в запросах (app/Http/Requests/Chats) с общим трейтом, системные логи вынесены в события (app/Events/Chats), а логика чатов разделена на ChatController и MessageController.
